# Quantron-test-work


Установка проекта
```sh
$ npm install
```

Запуск gulp watcher
```sh
$ gulp
```

Доступ по адресу:
```sh
http://localhost:7788/
```

Gulp-watcher следит за изменением файлов .html и также .sass/.scss в папке sass/ .
